## Backend Forum API

**Example:**

- **.env**

```.env
# HTTP SERVER
HOST=localhost
PORT=5000

# POSTGRES
PGHOST=localhost
PGUSER=developer
PGDATABASE=forumapi
PGPASSWORD=supersecretpassword
PGPORT=5432

# TOKENIZE
ACCESS_TOKEN_KEY=ACCESS_TOKEN_KEY_GENERATE
REFRESH_TOKEN_KEY=REFRESH_TOKEN_KEY_GENERATE
ACCCESS_TOKEN_AGE=3000
```

